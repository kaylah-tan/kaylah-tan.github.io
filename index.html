Data Sources and Project Ideas

  1. Electric Vehicles in Washington State

    Description: This dataset contains information about electric vehicle purchases in Washington State. Maintained by the Department of Licensining, this dataset stores information about the car itself (i.e., make, model, year, etc), cars battery status, vehicle registered to, etc.

    Project Idea: The aim for this project is to conduct analysis based on electric vehicles, specifically focusing on battery levels and other information regarding whether electric vehicles are relatively in high-demand, and the functionality of the vehicle. .

  2. Housing Affordability in Pierce County, WA.

    Description: This dataset contains information about housing affordability index in Pierce County from 1998-2023. Maintained by the Runstad Center for Real Estate Studies, this dataset stores information about HAI, median price, mortgage rate, household income, etc.

    Project Idea: The aim for this project is to understand past trends in housing prices and to create a machine learning model that predicts the likihood of a post-graduate being able to afford a home in the Pierce County area in the near future.

  3. Women 15 to 50 Years Who Had a Birth in the Past 12 Months by Marital Status and Educational Attainment.

    Description: This dataset contains information about Women residing in Washington State who have recently had a birth in the past year. Maintained by the U.S. Census Bureau, this dataset contains information about the women's educational attainment and marietal status, with Women ranging frmo 15-50.

    Project Idea: The aim for this project is to connect with a larger idea of single mothers. This project will require another dataset containing information about single mother's socioeconomic status in addition to the dataset above.

  4. Child mortality-trends in the United States.

    Description: This dataset contains information about child mortality trends from 1900-2018. This dataset can be furthered analyzed than the additional analysis from the Center for Disease Control (CDC). For example, questions about race and ethnicity, ages of pregnant women, highest deaths in a year, etc.

    Project Idea:: The aim for this project is to understand child mortality trends in the United States. This is a national public health issue that is not portrayed or discussed on the media very often.

  5. Adidas v. Nike Comparison

    Description: This dataset contains information about Adidas and Nike clothing information. This dataset is synethic, obtained from Kaggle.

    Project Idea: The aim for this project is to have fun comparing different information for two highly comptetitive sport brands. Some possible data analysis can look at differences in pricing, differences in shoe color, copies of one another, etc.
